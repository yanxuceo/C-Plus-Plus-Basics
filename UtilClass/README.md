
## Installation and Test

### Ubuntu 

```bash
$> cd ~
$> git clone https://github.com/yanxuceo/C-Plus-Plus-Basics.git
$> cd C-Plus-Plus-Basics/UtilClass
$> git clone https://github.com/google/googletest.git
$> mkdir build && cd build
$> cmake ..
$> make
$> cd ../scripts
$> sh runTest.sh
```
